#Description
 Set up auxiliary script of dotfiles ex) vim, screen, zsh ..etc

#How To Setup
 1. Prepare the dotfiles in the following locations: dotfiles
 2. Run setup.sh

#Usage
    setup.sh [-fvnhV]

#Options
     -f : Force create symbolic link
     -n : Dry run
     -h : view Help
     -v : increase Verbosity
     -V : view Version
